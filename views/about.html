About Wikistack

Wikistack is a simple Wikipedia clone implemented with Node.js, Express, Sequelize, and PostgreSQL.

The following fabulous features have been implemented:

  • A homepage that lists all the pages in the Wiki
  • The ability to create new pages, and write them in Markdown.
  • Page search by tag (via the navbar search bar)
  • Tagging
  • An authors page listing all the users, and individual user pages listing all the articles authored by each user.
  • The ability to delete pages.
  • Open page editing

The following features have yet to be implemented:

  • Page search by content
  • Version history
  • Discussion tab for each page
Thanks for stopping by. Check back soon for additional updates and functionality!
